BoredGap Projesi - Planlama ve Notlar
=================================

Proje Genel Yapısı:
------------------
1. Ana Sayfa
   - Platform kartları
   - Navigasyon menüsü
   - Üyelik sistemi entegrasyonu

2. Teknoloji Stack'i:
   - Frontend: React.js + TypeScript
   - Backend: Node.js + Express.js
   - Veritabanı: MongoDB (local)
   - Stil: Tailwind CSS
   - State Management: Redux

3. Klasör Yapısı:
/bgproject
├── client/
│   ├── public/
│   └── src/
│       ├── components/
│       ├── pages/
│       ├── styles/
│       ├── utils/
│       └── platforms/ (her platform için ayrı klasör)
├── server/
│   ├── config/
│   ├── models/
│   ├── routes/
│   └── controllers/
└── docs/

4. Temel Özellikler:
- Modüler platform yapısı
- Merkezi kimlik doğrulama sistemi
- Responsive tasarım
- Platform bazlı tema sistemi

5. İlk Aşama Görevleri:
[✓] Proje planı oluşturma
[✓] Temel klasör yapısının kurulması
[✓] Gerekli paketlerin kurulumu
[✓] Ana sayfa tasarımının oluşturulması
[✓] Üyelik sisteminin kurulması

6. Güvenlik Önlemleri:
- JWT tabanlı kimlik doğrulama
- Input validasyonu
- Rate limiting
- CORS yapılandırması

7. Tamamlanan İşler:
- Frontend projesinin oluşturulması (React + TypeScript)
- Tailwind CSS entegrasyonu
- Temel sayfa yapısının oluşturulması
- Navbar bileşeninin oluşturulması
- Ana sayfa tasarımının tamamlanması
- Platform kartlarının tasarımı
- Backend projesinin oluşturulması
- MongoDB bağlantısının kurulması
- Kullanıcı kimlik doğrulama sisteminin kurulması

8. Sıradaki Görevler:
- Frontend'de kullanıcı kimlik doğrulama entegrasyonu
- İlk platform (UwuFufu) geliştirmesine başlanması
- Platform yönlendirme sisteminin kurulması
- Kullanıcı profil sayfasının oluşturulması

9. API Endpoint'leri:
POST /api/users/register - Kullanıcı kaydı
POST /api/users/login - Kullanıcı girişi
GET /api/users/me - Kullanıcı bilgileri (kimlik doğrulama gerekli)

Not: Her platform bağımsız modül olarak geliştirilecek ve ana sisteme entegre edilecektir.
Her değişiklik bu dosyada güncellenecektir.
